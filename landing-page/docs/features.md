# Features

A complete reference of everything the extension can do.

## Blur Mode

Test yourself by blurring one or both subtitle rows. Click the eye icon in the control bar to choose a mode:

<img src={require('@site/static/img/features/blur-mode.png').default} alt="Blur mode" style={{maxWidth: '100%', borderRadius: '8px', border: '1px solid #404040'}} />

Hover over a blurred subtitle to reveal it temporarily.

## Word Lookup

`Select text in a subtitle` and `right click` to get an instant in-page lookup without leaving the video.
This will give you a word-by-word translation, suitable for beginners who cannot guess the meaning based on context yet.

<img src={require('@site/static/img/features/word-lookup-open.png').default} alt="Word lookup popup open" style={{maxWidth: '100%', borderRadius: '8px', border: '1px solid #404040'}} />

<img src={require('@site/static/img/features/word-lookup-explanation.png').default} alt="Word lookup popup explanation" style={{maxWidth: '100%', borderRadius: '8px', border: '1px solid #404040'}} />

## Rewind & Forward

Jump back or forward 3 seconds with the rewind/forward buttons in the control bar, or use keyboard shortcuts:

- `,` (comma) — rewind 3 seconds
- `.` (period) — forward 3 seconds

Useful for shadowing or active recall: repeat a sentence either by reading or by memory.

<img src={require('@site/static/img/features/rewind-forward.png').default} alt="Rewind and forward buttons" style={{maxWidth: '100%', borderRadius: '8px', border: '1px solid #404040'}} />

## Copy Finnish Subtitle

Click the copy icon to send the current Finnish subtitle line to your clipboard — handy for saving phrases. Personally I usually paste the text into AI chatbot for further explanation.

<img src={require('@site/static/img/features/copy-subtitle.png').default} alt="Copy Finnish subtitle" style={{maxWidth: '100%', borderRadius: '8px', border: '1px solid #404040'}} />

## Multi-Language Support

Translate Finnish subtitles into any language DeepL supports — English, Vietnamese, Japanese, Spanish, and 30+ others. Change your target language in the options page at any time (a page reload is required to apply the change).

<img src={require('@site/static/img/features/multi-language.png').default} alt="Multi-language support" style={{maxWidth: '100%', borderRadius: '8px', border: '1px solid #404040'}} />

## Subtitle Size

Use **Ctrl +** / **Ctrl −** in the browser to scale subtitles up or down. YLE Areena also has its own subtitle style settings (accessible via the speech-bubble icon in the player) where you can change font, size, and color. Check `Taustaväri` (Background color) and `Tekstityksen koko` (Subtitle size)

<img src={require('@site/static/img/features/subtitle-size.png').default} alt="Subtitle size adjustment" style={{maxWidth: '100%', borderRadius: '8px', border: '1px solid #404040'}} />

## Reload Subtitles

If translations look off, click the reload icon to clear the cached translations for the current episode and re-translate from scratch. This uses additional DeepL credits but can improve accuracy.

<img src={require('@site/static/img/features/reload-subtitles.png').default} alt="Reload subtitles" style={{maxWidth: '100%', borderRadius: '8px', border: '1px solid #404040'}} />

## Privacy

All data stays in your browser. No analytics, no tracking, no external servers beyond the DeepL API call itself.
